# **App Name**: LeafyAI

## Core Features:

- Image Upload: Allow users to upload leaf images directly from their device or using the camera.
- AI-Powered Disease Prediction: Utilize a pre-trained machine learning model to predict the likelihood of a leaf disease based on the uploaded image; provides a confidence score of the prediction.
- Display Results: Show the predicted disease or confirmation of a healthy leaf, accompanied by the confidence score.
- Treatment Suggestion Tool: If a disease is predicted, the application uses a tool to suggest potential treatments. 

## Style Guidelines:

- Primary color: Vibrant green (#6AB475), symbolizing health and nature.
- Background color: Light desaturated green (#EBF5EB), for a natural and calming backdrop.
- Accent color: Earthy brown (#A68B60) to highlight important UI elements, such as CTAs or alerts.
- Body and headline font: 'PT Sans' (sans-serif) to create a clean and readable user interface.
- Use leaf-related icons throughout the interface to maintain thematic consistency.
- Prioritize a clean and intuitive layout with clear visual hierarchy, emphasizing the image upload and result display areas.
- Incorporate subtle animations during the image processing and result display to enhance user engagement.
